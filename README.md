# CondoManager.Common

Core shared libraries for the CondoManager system.

## 📂 Modules

- **CondoManager.Entity** - Domain entities, DTOs, and enums  
- **CondoManager.Repository** - Data access layer with Entity Framework  

## 🔧 Usage

This repository is not distributed via NuGet.  
Instead, it is included as a **Git submodule** in the CondoManager Projects.
